NGX Ratio Bar

An Angular component that displays a unit interval value as a percentage.

Installation

Features

  • Show a custom label on the bar instead of the percentage value.
  • Choose to display a label at the end of the bar's footer.
  • An optional border can be added around the bar.
  • Bar foreground and background colors are customizable using property binding.

Usage

Import NgxRatioBarModule in the root module.

Then use it in your templates as follows:

Labels

To override the displayed percentage value with a custom label, simply set the ratio to any string value.

In addition, you can set the endLabel property, if you wish to display a label at the bottom-end of the bar.
